UpperCamelCase: Classes, enum types, typedefs, and type parameters should capitalize the first letter of each word (including the first word), and use no separators.
lowercase_with_underscores: name packages, directories, and source files, name import prefixes
lowerCamelCase: Class members, top-level definitions, variables, parameters, and named parameters should capitalize the first letter of each word except the first word, and use no separators.
